Understanding Data Flow

In this interactive vlog, we explore the three primary types of data flow in internet technology. Each type has distinct characteristics and use cases in modern networking.

Data flow refers to the movement of digital information between devices in a network. The efficiency and capabilities of these flows determine how we communicate, share information, and access services online.

Select a data flow type from the right to learn more and see an animation of how it works!

→

Simplex

One-way communication where data flows in only one direction. Examples include keyboard to computer, television broadcasting, and radio transmission.

↔

Half-Duplex

Two-way communication but only one direction at a time. Examples include walkie-talkies, traditional two-way radios, and some network protocols.

⇄

Full-Duplex

Simultaneous two-way communication where data can flow in both directions at the same time. Examples include telephone conversations and modern Ethernet networks.
